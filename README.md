# Mario Shooter
A 2d two player shooter game created with Gosu in 2 hours.


## CONTROLS
### Red Mario (Player 1)
W: Jump

A: Move left

D: Move right

SPACE: Shoot a fireball


### Blue Mario (Player 2)
Up arrow: Jump

Left arrow: Move Left

Right arrow: Move right

Enter/Return: Shoot a fireball 


### OTHERS
Press 'C' to play again
